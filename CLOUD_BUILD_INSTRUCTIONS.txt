云端构建APK使用说明
==================

由于本地构建环境存在一些问题，推荐使用GitHub Actions进行云端构建。

步骤：
1. 将smartclip_lite文件夹中的所有文件推送到GitHub仓库
2. GitHub Actions会自动开始构建APK
3. 构建完成后，可以在Actions页面下载生成的APK文件

GitHub Actions配置文件位置：
.github/workflows/android.yml

构建过程大约需要15-30分钟，生成的APK文件会自动上传为构建产物。

优点：
- 无需配置本地Android开发环境
- 避免网络和权限问题
- 使用标准的Ubuntu构建环境
- 自动化构建过程

如果需要帮助设置GitHub仓库，请联系开发者。

注意：构建过程会自动安装所有必要的依赖，包括Android SDK和NDK。
